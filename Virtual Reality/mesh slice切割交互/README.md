# mesh交互切割模型
实现下列部分功能  
（1）平面切割；（2）弧面切割；（3）局部切割（划口）；（4）区域切割；（5）打孔；（6）孔道。  
此项目工程所用Unity版本号为Unity 2019.4.21f1c1 (64-bit)
